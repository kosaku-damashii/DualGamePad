# DualGamePad

「Dual Game Pad」は、MachiKania type P開発基板やオレンジピコショップで発売されているORANGE-BUSへ接続できる外部コントローラです。

Dual Game Padの特徴
・ケーブルの取り付け方を変えることで、十字キーを右側にも左側にもすることが出来ます。
・コントローラからリセットすることが出来ます(2x5ピン接続時のみ)。
・基板上のフリーエリアに部品を追加することができます。
